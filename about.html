About us

Who we are

Green Mountain Web is a web development business led by Isaac Krementsov and Kaiden Cook, enthusiastic programmers. It seeks to further the development of businesses by partnering with them and displaying them online. See our GitHub at https://github.com/krementcook.

What we do

We use a varied set of tools in order to efficiently create a quality product. Our chief
back-end technology is NodeJS, which allows us to create fast, scalable web applications quickly. As for database storage, MongoDB is the primary strategy. Its document-based schemas allow complex data-structures and easy querying in NodeJS. Of course, our front-end uses the standard combination of CSS, HTML, and JavaScript, with Embedded JavaScript as a server-side templating engine.
